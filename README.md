# echo
An echo bot in multiple languages to get you started.

Direct:

- [node.js](./nodejs)
- [Rust](./rust)

With abstraction layer / bot framework:

- [bot-base (node.js)](./nodejs_bot_base)

The bot just echos your text messages when you send them to it as DM.
